
====== 在Windows下搭建React Native Android开发环境 ======

1、Node.js 安装配置：【https://nodejs.org/en/download/current/】

2、npm安装：在Node.js安装的时候顺带装好了。cmd：【npm -v】查看输出

3、安装淘宝npm（cnpm）：【npm install -g cnpm --registry=https://registry.npm.taobao.org】
npm config set registry https://registry.npm.taobao.org --global

npm config set disturl https://npm.taobao.org/dist --global

  1)、输入cnpm -v输入是否正常，如果报错请查看下面。
	
  2)、添加系统变量path的内容因为cnpm会被安装到D:\Program Files\nodejs\node_global下，而系统变量path并未包含该路径。在系统变量path下添加该路径即可正常使用cnpm。
  
4、进入terminal界面，输入【npm install】，如果失败，删除node_modules文件夹重新install。
