 ======== 基本命令 ========
 
1、日期：date 
         date +%Y年%m月%d日  【格式化日期】
		 
2、显示当前月份日历：cal 
                     cal 2018 【指定显示某年的日历】
					 cal [month] [year]  【指定显示某年某月的日历】
		 
3、重要的几个热键[Tab], [ctrl]-c, [ctrl]-d

4、ctrl + z  退出当前命令  相当于 quit 命令

5、ctrl + c  终止当前执行的命令

6、[Tab] 接在一串令的第一个字的后面，则为命令补全；
   [Tab] 接在一串令的第二个字以后时，则为『档案补齐』！

7、man page 【操作说明】  info info 【在线文档】
   代号
     1：用户在 shell 环境中可以操作的令戒可执行文件
	 5：配置文件戒者是某些档案的格式
	 8：系统管理员可用的管理令
	 
8、新建档案：【touch file】

9、新建目录：【mkdir folder】

10、增加新的群组：【groupadd project】

11、建立 account  账号，并支持project群组：【useradd -G project account 】

12、查阅 account  账号的属性：id account 

13、找出 /etc 底下，档案容量大亍 50K D档案所属人丌是 root m档名，D将权限完整m列出 (ls -
l)；find /etc -size +50k -a ! -user root -exec ls -ld {} \;
	find /etc -size +50k -a ! -user root -type f -exec ls -l {} \;

14、找出 /etc 底下，档案大小介亍 50K 到 60K Y间m档案，D将权限完整m列出 (ls -l)：
	find /etc -size +50k -a -size -60k -exec ls -l {} \;
	注意到 -a ，那个 -a 是 and m意忠，为符合两者才算成功
	
15、编辑器：【vim file.txt】，退出：【:wq+enter键】


 ======== 文件复制/移动/删除 ========
 
1、将档案的所以特性都一起复制过来:【cp -a /var/log/wtmp wtmp_2】

2、利用【cp -a /etc /tmp】来下达命令喔！尤其是在备份的情况下！

3、使用-l及-s都会建立所谓的连结档(link file)，那个-l是实体链接，-s 则是符号链接，bashrc_slink 是一个『杞莘绞健

  /tmp 可以将多个数据一次复制到同一个目录去！最后面一定是目！

6、如果没有加上任何选项时cp复制的是源文件，而非链接文件的属性！若要复制链接文件的属性，就得要使用 -d的选项了！

7、-r ：递归持续复制，用于目录的复制行为；(常用)

8、移动文件：【mv mkreleasehdr.sh redis-benchmark redis-check-aof redis-check-dump redis-cli redis-server /usr/local/bin/】

9、重命名：【mv file1 file2】

9、删除文件：【rm file】 删除文件夹：【rm -rf folder】


 ======== 查看进程 ========
 
1、关键字查看：【ps -ef | grep redis】 ps -ef | grep 6379】

2、端口号查看：【netstat -tunpl | grep 6379】 【netstat -ano | grep 6379】 

3、查看所有进程：【netstat -ntlp】
